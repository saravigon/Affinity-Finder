# MAIN/JAVA
Summary: main/java is a folder containing multiple folders where the whole files .java programmed are found.

## Main.java
This is the main entry point of the application. It initializes the application and starts the necessary components.

## domain
This folder contains all the domain classes that represent the core business logic and entities of the application.

## persistence
This folder contains all the classes responsible for data storage and retrieval, including database access and data management

## presentation
This folder contains all the classes related to the presentation layer of the application, including user interface components
and controllers.
