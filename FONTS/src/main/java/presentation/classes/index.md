# Presentation Layer Classes

This directory contains Java classes related to the presentation layer of the application. These classes are responsible for managing the user interface and handling user interactions. They may include components such as views, controllers, and UI utilities that facilitate the display of data and capture of user input.

## Key Classes

- **MainView.java**: The primary view class that serves as the main window of the application. It initializes and organizes other UI components.
- **InitialMenuView.java**: A view class that represents the initial menu presented to users upon launching the application. It provides navigation options to different sections of the application.

## Subdirectories
- **admin/**: Contains classes related to the administrative functionalities of the presentation layer.
- **profile/**: Contains classes that manage user profile views and interactions.
- **form/**: Contains form classes used for data input and validation within the presentation layer.
- **affinityGroups/**: Contains classes related to the presentation of affinity group features.
