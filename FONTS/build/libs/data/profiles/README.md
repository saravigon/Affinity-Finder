# PROFILE STORAGE

This folder contains user profile data used to customize and enhance your experience. Each profile is stored in its own file, named after the profile's unique identifier.

### File name:
`<UserID>.json`


### Example structure:
```json
{
  "description": "No description yet",
  "createdForms": [],
  "answeredForms": [],
  "UUID": -489870637,
  "user": "createdUser",
  "password": "pass"
}
```
